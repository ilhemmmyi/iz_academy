# Gilroy Font Files

Place the following font files in this directory:

| File | Weight |
|---|---|
| Gilroy-Light.woff2 | 300 |
| Gilroy-Light.woff | 300 |
| Gilroy-Regular.woff2 | 400 |
| Gilroy-Regular.woff | 400 |
| Gilroy-Medium.woff2 | 500 |
| Gilroy-Medium.woff | 500 |
| Gilroy-SemiBold.woff2 | 600 |
| Gilroy-SemiBold.woff | 600 |
| Gilroy-Bold.woff2 | 700 |
| Gilroy-Bold.woff | 700 |
| Gilroy-ExtraBold.woff2 | 800 |
| Gilroy-ExtraBold.woff | 800 |

Gilroy is a commercial font by Radomir Tinkov.
Purchase or obtain the font files and drop the .woff2 / .woff variants here.
The CSS @font-face declarations in src/styles/fonts.css are already wired up.
